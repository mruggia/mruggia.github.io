KeyBot (2008)

In some video games it can be beneficial to constantly mash a certain button. That's not fun. KeyBot is an application to automate that task, with up to 3 "mashers" and user selectable hotkeys and speeds. This program employs a lot of very questionable coding practices, since it is one of my first efforts in coding, but it still appears to be functional.

Biology Quiz (2009)

In the second year of high school my biology teacher wanted the class to learn the Latin names of some 40 common plants and how to recognize them. That seemed like a quite boring task, so instead I spent a lot of time programming an app to quiz me. It works by presenting a random image of a plant on screen, asking the user for its species/family, and correcting the answer if necessary. The database of names and images is managed with a separate program. By the time I was done programming, my twin sister Alessandra who also went to class with me, was already long done memorizing.. But that's the way things go when automating tasks.

DRS3 Player (2012)

At a time where the Swiss radio station SRF3 was still called DRS3 and there were only very rudimentary players, I decided to make my own. This application can play live radio and fetch metadata from a m3u stream, from which it can then find the appropriate cover art to display. Also implemented are tabs to listen to past DRS3 podcasts and to the hit-parade. While the UI still looks somewhat modern, it is the classical Windows Media Player waveform that gives away the year this was made in.

BF4 Lamp Solver (2016)

In Battlefield 4, a first-person-shooter I used to play a lot, there is a puzzle with 20 lamps and 7 switches. Each switch flips the state of some of the lamps and the goal of the puzzle is to find the switch positions that result in all lamps being on. This program solvers the puzzle and offers a somewhat intuitive UI to input which switches affect each lamp and to see the effect of various switch positions. I ended up using this program only two times to get the games reward for me and my friend Nicolo (the reward being a skin for the player character).
